# Deepfake Detection System

## Overview
This project detects deepfake content using machine learning.

## Features
- Detects fake or real content
- Supports video, image, audio, and live camera input
- Simple frontend interface

## Files
- app.py → Main execution
- detector.py → Detection logic
- frontend.py → User interface

## Note
Large files like trained models and videos are not included due to size limits.
